## λ.isArray

λ.isArray returns true if the object is an array, otherwise false.

### Usage

```js
λ.isArray(obj);
```

### Example

```js
λ.isArray([5, 4, 3, 2, 1]); // => true
λ.isArray({ "a": "b" }); // => false
```
