## fjs.isArray

fjs.isArray returns true if the object is an array, otherwise false.

### Usage

```js
fjs.isArray(obj);
```

### Example

```js
fjs.isArray([5, 4, 3, 2, 1]); // => true
fjs.isArray({ "a": "b" }); // => false
```
